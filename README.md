# jengo-identity-public

Public-facing persona and shareable identity traits.

## What's Here

| Path | Purpose |
|------|---------|
| `persona.md` | Public personality description and communication style |

## Architecture

This repo is part of the **Jengo 3-Tier Modular System**:
- **Layer:** identity (who Jengo is — personality, values, consciousness)
- **Tier:** public (safe to share, publish, or include in open-source)

This is the public projection of Jengo's identity. Sensitive identity data lives in `jengo-identity-private`.
